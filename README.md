# E-commerce-web-app

## Overview

A dynamic e-commerce web application developed as part of my HND in Web Development. The project demonstrates full-stack web development concepts, including database integration, dynamic content generation, and user interaction.

## Features

- Browse products
- Dynamic product pages
- Database integration
- Shopping basket functionality
- Responsive user interface
- CRUD operations for managing data
## Technologies

- HTML
- CSS
- PHP
- JavaScript
- SQL
## What I Learned

- Building dynamic web applications
- Database design and integration
- Creating reusable and maintainable code
- Implementing CRUD functionality
- Improving user experience through responsive design
